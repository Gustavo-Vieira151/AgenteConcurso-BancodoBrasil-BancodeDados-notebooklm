# AgenteConcurso-BancodoBrasil-BancodeDados-notebooklm
Um segundo cérebro especializado em Banco de Dados para concursos de TI,  reunindo teoria, questões comentadas, resumos e exemplos práticos,  com o objetivo de facilitar o aprendizado,  acelerar as revisões e aumentar o desempenho nas provas.

Tópico 1- NotebookLM + IA: Transformando Banco de Dados em um Guia de Estudos

"Estudar fica muito mais produtivo quando você faz as perguntas certas."

Tópico 2-  Sobre o projeto

Este projeto foi desenvolvido como parte do desafio da DIO utilizando o NotebookLM.

Ao invés de escolher um tema aleatório, resolvi aproveitar a oportunidade para estudar um assunto que realmente faz parte da minha rotina: Banco de Dados, um dos conteúdos cobrados no concurso Banco do Brasil – Agente de Tecnologia.

A ideia foi simples: reunir materiais de qualidade, deixar a Inteligência Artificial me ajudar a organizar o conteúdo e transformar tudo isso em um guia de estudos que eu possa consultar sempre que precisar.

Durante todo o processo utilizei o NotebookLM para interpretar as fontes e o ChatGPT para discutir ideias, criar estratégias de estudo e melhorar a organização do material.

Tópico 3 - Objetivo

Mais do que concluir um desafio, meu objetivo foi criar um material útil para os meus estudos.

Com o apoio da IA, consegui:

- organizar diferentes fontes em um único lugar;
- criar perguntas estratégicas para revisar o conteúdo;
- gerar resumos objetivos;
- montar um glossário dos principais conceitos;
- produzir prompts reutilizáveis para continuar estudando futuramente.

No fim, o projeto virou uma ferramenta de revisão para minha preparação.

Tópico 4 - Tema escolhido

Banco de Dados voltado para o concurso Banco do Brasil – Agente de Tecnologia.

Entre os assuntos estudados estão:

Modelagem de Banco de Dados
-  Como utilizei a IA

A proposta não foi simplesmente pedir respostas prontas.

Utilizei a IA como uma parceira de estudos:

- organizei as fontes no NotebookLM;
- fiz perguntas sobre os conteúdos;
- comparei diferentes formas de explicar os mesmos conceitos;
- solicitei resumos;
- pedi exemplos práticos;
- gerei questões para testar meu conhecimento;
- refinei prompts para obter respostas cada vez mais úteis.

Essa interação tornou o estudo muito mais dinâmico do que apenas ler PDFs.

Tópico 6- O que aprendi

Além de revisar Banco de Dados, percebi que a IA pode ser uma excelente ferramenta para organizar o conhecimento.

Em vez de substituir o estudo, ela ajudou a:

encontrar informações mais rapidamente;
conectar conceitos;
revisar conteúdos de forma mais eficiente;
criar materiais personalizados para futuras revisões.

No fim das contas, quem aprende continua sendo a pessoa. A IA apenas acelera esse processo quando é utilizada de forma consciente.


Tópico 7 - Resultado

Este repositório reúne o material produzido durante o desafio e também servirá como apoio para minha preparação para concursos na área de Tecnologia da Informação, especialmente para o Banco do Brasil – Agente de Tecnologia.

Se este projeto mostrar uma coisa, é que uma boa combinação de curiosidade, organização e Inteligência Artificial pode transformar horas de leitura em um estudo muito mais direcionado.
